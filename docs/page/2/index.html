3D重建-Fusiello立体校正

发表于 2023-10-10 更新于 2023-10-12 分类于 CV

本文介绍Fusiello (2000)立体校正算法[1]。

概述

立体校正的目的
立体校正也称为极线校正。通常双目相机用于获取深度。立体校正使得双目相机的图像的极线平行且和相机基线平行，且极线所在的行具有相同的行坐标，使得左极线上的点必在右极线所在图像行上，即匹配点直接通过行索引确定，也就是由普通双目相机变为理想双目相机，从而大幅度减小立体匹配的搜索空间。

立体校正后极线水平


极线校正后，匹配点像素y坐标相同。

Fusiello极线校正算法概述
算法要求：双目相机已标定，即已知相机内参、相互位置关系（旋转和平移）。
算法输入：两个相机的透视投影矩阵(世界点直接映射为图像点)。
算法输出：一对校正投影矩阵。

校正相机矩阵

假定双目相机已经标定，也就是透视投影矩阵PPM即P~o1,P~o2\tilde P_{o1},\tilde{P}_{o2}P~o1​,P~o2​已知。立体校正就是定义新的PPM P~n1,P~n2\tilde{P}_{n1},\tilde{P}_{n2}P~n1​,P~n2​，即通过旋转相机使得焦平面共面且包含基线，极线和基线平行。为使得极线水平，必须使基线平行于两个新的相机坐标系的X轴。此外要求匹配点必须有相同的垂直坐标，这需要使新的相机具有相同的内参。

也就是新相机的位置（光心）和旧相机相同，新相机的方向（旋转）和旧相机之间存在合适的旋转关系，新相机的内参相同。也就是新相机除了光心不同，其余相同，可以看做单个相机沿着其X轴平移得到。

基变换和旋转矩阵

发表于 2023-10-10 分类于 CV

摘要：推导坐标系旋转涉及的旋转变换矩阵，以及旋转矩阵和基的关系。

原理推导

从一个坐标系变换到另一个坐标系，通常使用基变换。
假设两个坐标系之间只有旋转关系，即原点重合，如下图

考虑坐标系O−WO-WO−W中一点P，在世界坐标系中的坐标表示为Pw=(xw,yw,zw)TP_w=(x_w,y_w,z_w)^TPw​=(xw​,yw​,zw​)T，世界坐标系的标准正交基为B1=(e1,e2,e3)B_1=(\bm e_1, \bm e_2, \bm e_3)B1​=(e1​,e2​,e3​)，点P在O−CO-CO−C坐标系的坐标表示为Pc=(xc,yc,zc)TP_c=(x_c,y_c,z_c)^TPc​=(xc​,yc​,zc​)T，且该坐标系的标准正交基为B2=(b1,b2,b3)B_2=(\bm b_1, \bm b_2, \bm b_3)B2​=(b1​,b2​,b3​)，根据坐标的定义，有

p=OP⃗=B1Pw=B2Pc\bm p=\vec{OP}=B_1P_w=B_2P_c p=OP=B1​Pw​=B2​Pc​

可得

Pc=B2−1B1PwP_c=B_2^{-1}B_1P_w Pc​=B2−1​B1​Pw​

令

R=B2−1B1R=B_2^{-1}B_1 R=B2−1​B1​

是正交矩阵，则有Pc=RPwP_c=RP_wPc​=RPw​。即点PPP在不同坐标系下的坐标表示可由旋转进行变换得到。

考虑到bi\bm b_ibi​可由B1B_1B1​线性表示，并且wi\bm w_iwi​表示该基向量在O−WO-WO−W坐标系下的坐标，则

bi=(e1,e2,e3)wi=B1wi\bm b_i=(\bm e_1, \bm e_2, \bm e_3)\bm w_i=B_1\bm w_i bi​=(e1​,e2​,e3​)wi​=B1​wi​

即

B2=(e1,e2,e3)[w1,w2,w3]=B1WB_2=(\bm e_1, \bm e_2, \bm e_3)[\bm w_1, \bm w_2, \bm w_3]=B_1W B2​=(e1​,e2​,e3​)[w1​,w2​,w3​]=B1​W

即

B1Pw=B2Pc=B1WPcB_1P_w=B_2P_c=B_1WP_c B1​Pw​=B2​Pc​=B1​WPc​

可得

Pc=W−1PwP_c=W^{-1}P_w Pc​=W−1Pw​

即

R=W−1=B2−1B1R=W^{-1}=B_2^{-1}B_1 R=W−1=B2−1​B1​

当B1B_1B1​取单位矩阵时，R=B2−1=B2TR=B_2^{-1}=B_2^TR=B2−1​=B2T​。

应用

在相机成像过程中，通常会涉及到坐标系旋转、相机外参矩阵等问题，当世界坐标系的基取单位阵的列向量时，世界坐标系到相机坐标系的旋转可由相机坐标系的基确定。同样，知道相机坐标系的基，可以得到旋转矩阵。这一点在Fusiello立体校正[2]中使用到。

3.2 透视现象

主要两个表现：

  • 近大远小现象（沿光轴方向）
  • 偏离光轴越远，畸变越大（平行于图像平面方向，垂直光轴方向）。

广角畸变：距离镜头越近，畸变越大；偏离光轴越远，畸变越大。

透视畸变相关概念：消失点等概念


当拍摄一个目标时，只要它的表面存在物距上的差异，那么这个物体上每一个物距不同的局部区域都会有 不同的成像放大倍率。这种现象不管是通过镜头成像，还是小孔成像都是存在的，是固有的属性，这里的投影模型就叫做透视投影，其产生的效应我称之为透视效应。

近大远小解释：
由pc=sPcp_c=sP_cpc​=sPc​，有

{xc=fZcXcyc=fZcYc\begin{cases} x_c=\frac{f}{Z_c}X_c\\ y_c=\frac{f}{Z_c}Y_c \end{cases} {xc​=Zc​f​Xc​yc​=Zc​f​Yc​​

取相同高度的两个物体（物体高度Xc,YcX_c,Y_cXc​,Yc​），其对应的像素点坐标xc,ycx_c,y_cxc​,yc​距离近（即物距ZcZ_cZc​小）的值大。

偏离光轴方向的畸变原理：
偏离角度越大，Δx,Δy\Delta x,\Delta yΔx,Δy越大。（正切，同一物距下）

视觉Transformer (ViT)入门

发表于 2023-06-18 分类于 CV

Transformer最早用于NLP领域中，视觉Transformer（Visual Transformer，ViT）出自Google的论文 An Image is Worth 16x16 Words: Transformers for Image Recognition at Scale, ICLR 2021，是基于Transformer 模型在视觉领域的开篇之作。本文主要介绍其网络结构和基本原理。

摘要

虽然Transformer体系结构已经成为自然语言处理任务的事实标准，但其在计算机视觉中的应用仍然有限。在视觉中，注意力要么与卷积网络结合使用，要么用于替换卷积网络的某些组件，同时保持其整体结构。我们证明，这种对CNN的依赖是不必要的，并且直接应用于图像块序列的纯变换器（Transformer）可以在图像分类任务中表现得很好。 当在大量数据上进行预训练并转移到多个中小型图像识别基准任务（ImageNet、CIFAR-100、VTAB等）时，与最先进的卷积网络相比，视觉Transformer获得了优异的结果，同时需要更少的计算资源进行训练。

基于自注意力结构，Transformer已成为NLP领域的首选方法。由于其计算效率和可扩展性（scalability），使得可以训练更大规模的网络，随着模型和数据的规模增长，仍然没有饱和的迹象。

在计算机视觉中，卷积架构仍然占据主导地位。许多研究受NLP中的成功启发，尝试在CNN类结构结合自注意力机制，但是表现并不好。在大规模图像分类中，经典的ResNet类网络结构仍然是效果最好的。

受Transformer扩展到NLP领域的成功启发，我们使用了把标准Transformer仅做小改动后直接应用到图像，但是结果令人沮丧。我们把图像分块，并对块生成特征（线性embedding）后送入Transformer，图像块按照NLP中token那样使用。在图像分类任务中使用监督训练，在中等规模数据集如ImageNet上训练，但是模型结果明显不如ResNet类网络的结果。

和本文最相关的研究是 On the relationship between selfattention and convolutional layers. ICLR, 2020，其提取2x2大小的图块，并使用全自注意力，其只适合于小分辨率图像。

我们认为这种结果是意料之中的，因为Transformer缺乏CNN网络的归纳性偏好（the inductive biases），因此不能在训练数据不充足的情况下泛化好。

但是在更大规模数据集（14M-300M图片）上训练后情况变得不同，此时可以胜过归纳偏好。当在充足的大规模数据集上训练，再迁移到小数据集上时，ViT获得极好的结果。

模型结构

标准Transformer输入token的一维序列，对于2D图像，将图像划分为展平块的序列，即CCC通道的图像x∈RH×W×Cx\in \mathbb{R}^{H\times W\times C}x∈RH×W×C变为xp∈RN×(P2C)x_p\in \mathbb{R}^{N\times (P^2C)}xp​∈RN×(P2C)。即原始图像的分辨率为H×WH\times WH×W，通道数为CCC，分块大小为P×PP\times PP×P，总块数N=HW/P2N=HW/P^2N=HW/P2。整个Transformer使用隐藏（latent）向量大小DDD，因此我们用可训练的MLP线性投影来展平patch，把该投影的输出称为patch embeddings。如公式(1)所示：

其中LN：LayerNorm，MSA：Multihead self-attention。

在原有输入序列前加入一个可学习的特征 xclassx_{\rm class}xclass​，类似于BERT。（为执行分类，使用标准方法即给序列添加一个额外的分类token）

使用了标准的可学习的一维位置编码，因为没有发现2D位置编码带来显著的性能提升。

归纳偏好 我们发现视觉Transformer比CNN有更少的视觉上的归纳偏好。CNN中，局部性、二维邻域结构和平移不变性（translation equivariance）被结合到每一层并遍及整个模型。但是ViT中，只有MLP层是局部和平移不变的，尽管自注意力层是全局的。二维邻域结构被使用得很少：在模型开始前将图像划分为图块时或者fune-tuning时调整位置编码以应对不同分辨率的图像。此外位置编码在初始化时未携带任何2D图块位置信息，所有空间关系都是从头学习得到的。

混合结构 除了使用原始图像块，可以将CNN输出的特征作为模型输入，即patch embedding projection模块可以将CNN输出的特征图进行映射。

Fine-tuning和高分辨率

通常我们在大数据集上预训练ViT，然后在小的下游任务上微调。为此我们移除前期的预测模块，并连接一个零初始化的D×KD\times KD×K的前馈网络，KKK是下游任务的类别数。

对于大分辨率图像，我们保持块大小相同，这导致更大的序列长度（即N的大小），ViT支持任意大小的序列长度（不考虑内存约束）。对于大分辨率图像，位置编码不再有效，此时我们对预训练的位置编码进行2D插值。
