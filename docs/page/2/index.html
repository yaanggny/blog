立体校正目的

立体校正，也称为极线校正。目的是使得极线和双目相机的基线平行，从而使得匹配点的垂直坐标相同，即极线和扫描线平行，使得匹配点可直接由行号确定，从而大幅减小立体匹配的搜素计算量。

发表于 2023-10-10 更新于 2023-10-12 分类于 CV

本文介绍Fusiello (2000)立体校正算法[1]。

概述

立体校正的目的
立体校正也称为极线校正。通常双目相机用于获取深度。立体校正使得双目相机的图像的极线平行且和相机基线平行，且极线所在的行具有相同的行坐标，使得左极线上的点必在右极线所在图像行上，即匹配点直接通过行索引确定，也就是由普通双目相机变为理想双目相机，从而大幅度减小立体匹配的搜索空间。

立体校正后极线水平

Fusiello极线校正算法概述
算法要求：双目相机已标定，即已知相机内参、相互位置关系（旋转和平移）。
算法输入：两个相机的透视投影矩阵(世界点直接映射为图像点)。
算法输出：一对校正投影矩阵。

摘要：推导坐标系旋转涉及的旋转变换矩阵，以及旋转矩阵和基的关系。

原理推导

从一个坐标系变换到另一个坐标系，通常使用基变换。
假设两个坐标系之间只有旋转关系，即原点重合，如下图

考虑坐标系O−WO-WO−W中一点P，在世界坐标系中的坐标表示为Pw=(xw,yw,zw)TP_w=(x_w,y_w,z_w)^TPw​=(xw​,yw​,zw​)T，世界坐标系的标准正交基为B1=(e1,e2,e3)B_1=(\bm e_1, \bm e_2, \bm e_3)B1​=(e1​,e2​,e3​)，点P在O−CO-CO−C坐标系的坐标表示为Pc=(xc,yc,zc)TP_c=(x_c,y_c,z_c)^TPc​=(xc​,yc​,zc​)T，且该坐标系的标准正交基为B2=(b1,b2,b3)B_2=(\bm b_1, \bm b_2, \bm b_3)B2​=(b1​,b2​,b3​)，根据坐标的定义，有

p=OP⃗=B1Pw=B2Pc\bm p=\vec{OP}=B_1P_w=B_2P_c p=OP=B1​Pw​=B2​Pc​

可得

Pc=B2−1B1PwP_c=B_2^{-1}B_1P_w Pc​=B2−1​B1​Pw​

令

R=B2−1B1R=B_2^{-1}B_1 R=B2−1​B1​

是正交矩阵，则有Pc=RPwP_c=RP_wPc​=RPw​。即点PPP在不同坐标系下的坐标表示可由旋转进行变换得到。

考虑到bi\bm b_ibi​可由B1B_1B1​线性表示，并且wi\bm w_iwi​表示该基向量在O−WO-WO−W坐标系下的坐标，则

bi=(e1,e2,e3)wi=B1wi\bm b_i=(\bm e_1, \bm e_2, \bm e_3)\bm w_i=B_1\bm w_i bi​=(e1​,e2​,e3​)wi​=B1​wi​

即

B2=(e1,e2,e3)[w1,w2,w3]=B1WB_2=(\bm e_1, \bm e_2, \bm e_3)[\bm w_1, \bm w_2, \bm w_3]=B_1W B2​=(e1​,e2​,e3​)[w1​,w2​,w3​]=B1​W

即

B1Pw=B2Pc=B1WPcB_1P_w=B_2P_c=B_1WP_c B1​Pw​=B2​Pc​=B1​WPc​

可得

Pc=W−1PwP_c=W^{-1}P_w Pc​=W−1Pw​

即

R=W−1=B2−1B1R=W^{-1}=B_2^{-1}B_1 R=W−1=B2−1​B1​

当B1B_1B1​取单位矩阵时，R=B2−1=B2TR=B_2^{-1}=B_2^TR=B2−1​=B2T​。

应用

在相机成像过程中，通常会涉及到坐标系旋转、相机外参矩阵等问题，当世界坐标系的基取单位阵的列向量时，世界坐标系到相机坐标系的旋转可由相机坐标系的基确定。同样，知道相机坐标系的基，可以得到旋转矩阵。这一点在Fusiello立体校正[2]中使用到。

3.2 透视现象

主要两个表现：

  • 近大远小现象（沿光轴方向）
  • 偏离光轴越远，畸变越大（平行于图像平面方向，垂直光轴方向）。

广角畸变：距离镜头越近，畸变越大；偏离光轴越远，畸变越大。

透视畸变相关概念：消失点等概念


当拍摄一个目标时，只要它的表面存在物距上的差异，那么这个物体上每一个物距不同的局部区域都会有 不同的成像放大倍率。这种现象不管是通过镜头成像，还是小孔成像都是存在的，是固有的属性，这里的投影模型就叫做透视投影，其产生的效应我称之为透视效应。

近大远小解释：
由pc=sPcp_c=sP_cpc​=sPc​，有

{xc=fZcXcyc=fZcYc\begin{cases} x_c=\frac{f}{Z_c}X_c\\ y_c=\frac{f}{Z_c}Y_c \end{cases} {xc​=Zc​f​Xc​yc​=Zc​f​Yc​​

取相同高度的两个物体（物体高度Xc,YcX_c,Y_cXc​,Yc​），其对应的像素点坐标xc,ycx_c,y_cxc​,yc​距离近（即物距ZcZ_cZc​小）的值大。

偏离光轴方向的畸变原理：
偏离角度越大，Δx,Δy\Delta x,\Delta yΔx,Δy越大。（正切，同一物距下）
