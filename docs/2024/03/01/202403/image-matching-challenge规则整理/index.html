发表于 2024-03-01 分类于 图像匹配

IMC-2023规则

image matching chanllenge 2023

竞赛目标

本次竞赛的目标是重建精确的3D地图。去年的比赛聚焦于两视图匹配。本次比赛更进一步，从多个不同视图重建3D场景。
你的工作可以成为从嘈杂的数据（例如用户上传到 Google 地图等服务的图像）中解锁世界地图的关键。

从图像集合重建环境的3D模型的过程被称为运动恢复结构（SfM）。这些图像通常由受过训练的操作员或使用额外的传感器数据拍摄，例如谷歌地图使用的汽车。这确保了同质、高质量的数据。考虑到各种各样的视角，以及光线、天气和其他变化，从各种图像构建3D模型要困难得多。

竞赛主办方谷歌在许多谷歌地图服务中使用了SfM技术，例如街景和航空图像创建的3D模型。为了加快对这一主题的研究，更好地利用已经公开的数据量，谷歌与Haiper和Kaggle合作举办了这场比赛。

你在帮助建立准确的3D模型方面的工作可能会应用于摄影、文化遗产保护和谷歌的许多服务。

评价方式

评价指标

参赛者被要求估计图像集合（N个图象）中每个图像的位姿。每个相机位姿参数化为一个旋转矩阵和平移向量，相对于任意的坐标系。

提交的位姿使用mean Average Accuracy (mAA) 评价。给定一组经过参数化（使用旋转和平移）相机，以及隐藏的真值（位姿）我们计算每个可能的图像对（？） 之间的旋转和平移误差，也就是共有CN2C_N^2CN2​对。

Problem Definition

竞赛目标是估计两个图像之间的相对位姿。这需要透视和对极几何的知识。

  • 标定矩阵确定3D点和2D（像素）坐标之间的变换，也称为相机内参。
  • 旋转矩阵RRR和平移向量TTT捕获相机在全局参考坐标系中的6自由度位姿（位置和方向），它们被称为相机外参。
  • 基础矩阵FFF概括了同一场景的两个视图之间的投影几何，其不受场景内容的影响，之却决于两个相机之间的内参和外参。

对极几何

对极几何是两个视图之间的投影几何，其不依赖于场景的结构或内容，之取决于相机内参和外参。

在计算机视觉中，这种关系表述为基础矩阵，一个稚为2的3x3矩阵。一个单位为米的3D点MiM_iMi​，其在两个不同的相机中的投影为mi,mi′m_i, m_i'mi​,mi′​（单位：像素），基础矩阵满足mi′TFmi=0m_i'^TFm_i=0mi′T​Fmi​=0。可以用一组稀疏对应关系来计算基础矩阵，如8点法，或者OpenCV、Kornia。

  • Multiple View Geometry in Computer Vision: The reference book on multiple-view geometry, with free chapters on epipolar geometry/fundamental matrix estimation.

评价指标

参与者被要求去估计两个图像之间的相对位姿。对估计位姿的评价使用平均精度 mean Average Accuracy (mAA)。给定一个基础矩阵和隐藏的真值，我们计算旋转和平移的误差(ϵR\epsilon_RϵR​,单位：度，ϵT\epsilon_TϵT​,单位：米)。给定两个阈值，如果一个位姿的旋转和平移都满足则认为该位姿精确。我们在10对阈值上评估（1度到10度，20cm到5m）:
