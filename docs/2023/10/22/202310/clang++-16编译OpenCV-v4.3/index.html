clang++-16编译OpenCV-v4.3

发表于 2023-10-22 更新于 2023-10-23 分类于 编译

记录基于clang++ 16.0.0 在Windows上编译OpenCV4.3.0的过程。

环境配置

使用Ninja工具。

CMake配置

  • 关闭DSHOW,webp,dnn，即不构建这些模块。否则会报错。
  • ippicv: 需要安装，可手动下载，放在OpenCV根目录下.cache目录下。具体根据错误提示处理。
  • CMAKE_BUILD_TYPE=Debug
