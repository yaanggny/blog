VirtualBox虚拟机技巧

发表于 2023-09-13 分类于 Linux

本博客主要记录VirtualBox虚拟机相关的技巧和经验。

VirtualBox 虚拟机和主机共享粘贴板

  • 选择虚拟机，然后选择设置，选择“常规”中的“高级”，将“共享粘贴板”的选项设置为双向，如下图。

  • 启动虚拟机
  • 启动后在“设备”菜单下选择“安装增强功能”
  • 然后点击“运行/Run”，输入密码安装。
  • 安装完毕后重启虚拟机。

和主机共享文件夹

原理：用户组概念。

  • 选择虚拟机，点击设置，选择共享文件夹
  • 点击右侧带加号的文件夹图标，创建路径和挂载点，如下图

  • 打开虚拟机，测试ls /mnt/common是否可列出文件目录。
    一般会提示无权限。
  • 添加当前用户到用户组。
  • 重启生效（必须重启）
