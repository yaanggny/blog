Linear-Attention

发表于 2023-06-03 更新于 2023-06-04 分类于 DL ， CV

线性Attention主要指其计算复杂度为 特征长度（序列长度）n的线性倍数，即O(n)O(n)O(n)，而不是Attention的实现是线性运算（乘法、加法）。标准Attention具有的平方复杂度。

线性Attention已经有很多改进，本文主要分析Linear Attention（2020），该线性Attention模型在LoFTR算法中使用。

Transformers are RNNs: Fast Autoregressive Transformers with Linear Attention（2020）

标准Attention

标准Attention，即Scaled Dot-production Attention，形式为

Attention(Q,K,V)=softmax(QKTdk)VAttention(Q,K,V)={\rm softmax}(\frac{QK^T}{d_k})V Attention(Q,K,V)=softmax(dk​QKT​)V

其中Q∈Rn×dk,K∈Rn×dk,V∈Rn×dvQ\in \mathbb{R}^{n\times d_k}, K\in \mathbb{R}^{n\times d_k}, V\in \mathbb{R}^{n\times d_v}Q∈Rn×dk​,K∈Rn×dk​,V∈Rn×dv​。softmax{\rm softmax}softmax函数逐行作用于矩阵QKTQK^TQKT。
其中

softmax(QKT)ij=eQi(KT)jT∑j=1neQi(KT)jT{\rm softmax}(QK^T)_{ij}=\frac{e^{Q_i(K^T)_j^T}}{\sum_{j=1}^n e^{Q_i(K^T)_j^T}} softmax(QKT)ij​=∑j=1n​eQi​(KT)jT​eQi​(KT)jT​​

式中矩阵带下标均表示矩阵的行向量，Attention输出第iii行jjj列为

Aij=∑k=1nsoftmax(QKT)ikVkjA_{ij}=\sum_{k=1}^n{\rm softmax}(QK^T)_{ik}V_{kj} Aij​=k=1∑n​softmax(QKT)ik​Vkj​

一个Transformer是一个函数T:Rn×d→Rn×dT:\mathbb{R}^{n\times d} \rightarrow \mathbb{R}^{n\times d}T:Rn×d→Rn×d， 由L个Transformer层T1(⋅),⋯ ,TL(⋅)T_1(\cdot),\cdots,T_L(\cdot)T1​(⋅),⋯,TL​(⋅)构成，其中

Tl(x)=fl(Al(x)+x)T_l(\bm x)=f_l(A_l(\bm x) + \bm x) Tl​(x)=fl​(Al​(x)+x)

fl(⋅)f_l(\cdot)fl​(⋅)独立地变换特征，通常由一个两层的前馈神经网络（MLP）构成。Al(⋅)A_l(\cdot)Al​(⋅)是自注意力函数。

自注意力函数的一般形式可以写成

Aij=∑k=1nsim(Qi,(KT)k)Vk,j∑j=1nsim(Qi,(KT)j)A_{ij}=\frac{\sum_{k=1}^n{\rm sim}(Q_i,(K^T)_k)V_{k,j}}{\sum_{j=1}^n{\rm sim}(Q_i,(K^T)_j)} Aij​=∑j=1n​sim(Qi​,(KT)j​)∑k=1n​sim(Qi​,(KT)k​)Vk,j​​

Ai=sim(Qi,KT)V∑j=1nsim(Qi,Kj)A_i=\frac{ {\rm sim}(Q_i,K^T)V}{\sum_{j=1}^n{\rm sim}(Q_i,K_j)} Ai​=∑j=1n​sim(Qi​,Kj​)sim(Qi​,KT)V​

  1. 论文中的公式应该有误，sim(⋅){\rm sim}(\cdot)sim(⋅)结果应该是标量，原始式子无法表达出行向量。
  2. 一些文章解读称“Scaled Dot-production Attention相当于以sim(⋅){\rm sim}(\cdot)sim(⋅)为权重对ViV_iVi​做加权平均”似乎不合适。

其中矩阵带下标均表示矩阵的一个行向量，要求sim(⋅){\rm sim}(\cdot)sim(⋅)作为注意力函数要非负。而softmax可以保证非负。

修改sim(⋅){\rm sim}(\cdot)sim(⋅)函数就可以实现不同的注意力函数。当sim(Qi,Kj)=exp⁡(Qi(KT)jTdk){\rm sim}(Q_i, K_j)=\exp\left(\frac{Q_i(K^T)_j^T}{\sqrt{d_k}}\right)sim(Qi​,Kj​)=exp(dk​​Qi​(KT)jT​​)，即为Scaled Dot-production Attention。

线性Attention

点积Attention定义是通用的，可以用作定义多项式attention和RBF核attention。给定核函数ϕ(x)\phi(x)ϕ(x)，注意力函数输出为

sim(Qi,(KT)j)=ϕ(Qi)⋅ϕ((KT)j)=ϕ(Qi)ϕ((KT)j)T{\rm sim}(Q_i,(K^T)_j)=\phi(Q_i)\cdot \phi((K^T)_j)=\phi(Q_i) \phi((K^T)_j)^T sim(Qi​,(KT)j​)=ϕ(Qi​)⋅ϕ((KT)j​)=ϕ(Qi​)ϕ((KT)j​)T

Aij=∑k=1nϕ(Qi)ϕ((KT)k)TVk,j∑j=1nϕ(Qi)ϕ((KT)j)T=ϕ(Qi)∑k=1nϕ((KT)k)TVk,jϕ(Qi)∑j=1nϕ((KT)j)TA_{ij}=\frac{\sum_{k=1}^n\phi(Q_i)\phi((K^T)_k)^TV_{k,j}}{\sum_{j=1}^n\phi(Q_i)\phi((K^T)_j)^T}=\frac{\phi(Q_i)\sum_{k=1}^n\phi((K^T)_k)^TV_{k,j}}{\phi(Q_i)\sum_{j=1}^n\phi((K^T)_j)^T} Aij​=∑j=1n​ϕ(Qi​)ϕ((KT)j​)T∑k=1n​ϕ(Qi​)ϕ((KT)k​)TVk,j​​=ϕ(Qi​)∑j=1n​ϕ((KT)j​)Tϕ(Qi​)∑k=1n​ϕ((KT)k​)TVk,j​​

根据矩阵乘法结合律，可以得到

(ϕ(Q)ϕ(K)T)V=ϕ(Q)(ϕ(K)TV)\left(\phi(Q)\phi(K)^T\right)V=\phi(Q)\left(\phi(K)^T V\right) (ϕ(Q)ϕ(K)T)V=ϕ(Q)(ϕ(K)TV)

注意ϕ(⋅)\phi(\cdot)ϕ(⋅)逐行作用于Q,KQ,KQ,K矩阵。

论文中的复杂度：通过上述修改，使用核函数替换softmax后，原有点积注意力的时间和空间复杂度均由O(n2)O(n^2)O(n2)降低为O(n)O(n)O(n)，其中nnn是输入特征维数，因为可以通过提前计算部分值减小计算量。

本文作者分析的复杂度：
具体考虑内部内积的循环次数，点积Attention的时间复杂度为O(n2d+nd∗n)=O(2n2d)O(n^2d+nd*n)=O(2n^2d)O(n2d+nd∗n)=O(2n2d)，线性Attention的时间复杂度为O(d2n+nd∗d)=O(2d2n)O(d^2n+nd*d)=O(2d^2n)O(d2n+nd∗d)=O(2d2n)。若线性Attention复杂度更小，至少要求d<nd<nd<n。
