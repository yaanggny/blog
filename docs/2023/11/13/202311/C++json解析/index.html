C++ json解析

发表于 2023-11-13 分类于 工具

本博客记录json解析库 nlohmann/json 的使用方法。

为什么选用这个库：

  • 单文件，无其他依赖
  • 易用，接口友好易上手（基于现代C++语法）
  • 较好的性能
  • 良好充分的测试（使用 Valgrind, Clang Sanitizers 测试，无内存泄漏）
