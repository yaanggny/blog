发表于 2023-12-08 更新于 2024-03-01 分类于 传感器标定

在Ubuntu虚拟机上实现禾赛激光雷达（AT128）调试

  • 在PandarView 2上获取实时点云
  • 用官方ROS驱动在ROS rviz中获取实时点云

环境设置

  • Virtual Box 6.1
  • Ubuntu 18.04
  • PandarView 2
  • https://github.com/HesaiTechnology/HesaiLidar_ROS_2.0 ROS驱动
  • https://github.com/HesaiTechnology/HesaiLidar_SDK_2.0 SDK
  • Wireshark

Windows 10上用PandarView 2中获取点云

按照用户手册进行网络设置，但是并不起作用，按照官方文档的设置没有调通。根据其他博客，原因是LiDAR的IP被修改过。但可以参考相关设置的概念。

基于多播IP+UDP端口的网络设置方法的步骤如下：

  • 用网线连接LiDAR到电脑，并上电（AT128等开机后便持续发送点云数据）。
  • Wireshark抓包，获取IP和端口

Wireshark打开后，选择相应网卡，然后就可看到类似该图。选择Len比较大的行，记录Dst IP和Dst Port。

  • 在PandaView 2中的监听对话框的Multicast IP和UDP Port栏分别填入上述IP和端口号。
  • 此时若没有报错，应该可以看到在一个平面上且很稀疏的点云。
  • 在PandaView 2中选择校正按钮，选择对应型号的角度校正文件。
  • 此时应该可以在PandarView 2中看到正常的实时点云。

VirtualBox Ubuntu虚拟机中用PandarView 2获取点云

经过尝试，Ubuntu虚拟机内部的PandarView 2软件设置同Windows。区别在于Ubuntu虚拟机的网络设置。需要改为以下设置：桥接模式且选择网线连接。
